### JS project set up
- `npm install -D @babel/core @babel/cli @babel/preset-env @babel/node`
- create a `.babelrc` file at the root of your project and add this - 
```
{"presets": ["@babel/preset-env"]}
```
- add start script
- add mocha and chai
- add test script
